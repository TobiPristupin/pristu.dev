Minimal Running

Minimal Running is an Android application that allows users to track their running, visualize it, and securely sync it with the cloud using an email or Google account. This is my largest project, and took me over a year to complete. Minimal Running taught me how to work with a large codebase, use version control, and effectively apply design patterns, as well as the nuances of deploying code to users all around the world. Find it on the Play Store.

PiAlarm / PiVoice

This is my personal alarm. It has two components to it, an alarm and voice control. Both are running on a Raspberry Pi inside Google's AIY Voice Kit. The alarm contains a Python program that periodically fetches my next alarm using Google's Calendar API and displays it on a 5 inch screen using Kivy for the GUI. The voice component uses Google's Voice Assistant API. Both work together to create a smart alarm that integrates Google Calendar, a GUI, and a voice assistant.

WaterMyCactus

I always forget to water my plants, so I built WaterMyCactus. It is based on an Arduino connected to a soil moisture sensor. When the soil moisture dips below a certain threshold, the arduino activates a water pump that waters the plant.
